			*** Git + Github ***







Comando: git add .

adiciona varios arquivos ao mesmo tempo na git bash

 

Comando: git  commit -m "commit inicial"


para criar novas versoes do proprio codigo

Comando: git remote add origin (https://github.com/Khaled-Klaus-Queiroz-de-Souza/projeto1_teste)

para adicionar uma pasta de origem na nuvem (juntamente com o link de onde vai ficar a pasta)

Comando: git push

para enviar o que esta sendo feito para o servidor da nuvem


Comando git push --set-upstream origin master

para identificar a parte principal na nuvem

Comando: git reflog 

para ver versões do codigo


Comando: git reset --hard (id que quer resetar a versão)

para resetar a versão do codigo tanto passado quanto futuro
